## Why Start a Blog?

Writing a blog means a lot to me for several reasons:

1. **Clarify thinking** — Turning scattered thoughts into words forces me to think clearly
2. **Track growth** — Looking back at old posts shows how much I've changed
3. **Share knowledge** — If it helps even one person, it's worth it

## Tech Stack

This blog is built with:

- **Next.js 15** — A powerful React framework
- **next-intl** — An elegant internationalization solution
- **Giscus** — Comments powered by GitHub Discussions
- **Tailwind CSS** — Writing styles has never felt this good
