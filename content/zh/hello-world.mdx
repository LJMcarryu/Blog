## 为什么写博客？

写博客对我来说有几个重要的意义：

1. **整理思路** — 把脑子里散落的想法写成文字，逼着自己想清楚
2. **记录成长** — 回头看以前写的东西，能感受到自己的变化
3. **分享知识** — 如果能帮到一个人，就值了

## 技术选型

这个博客用了以下技术：

- **Next.js 15** — 强大的 React 框架
- **next-intl** — 优雅的国际化方案
- **Giscus** — 基于 GitHub Discussions 的评论系统
- **Tailwind CSS** — 写样式从来没这么爽过
