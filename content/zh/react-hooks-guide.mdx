## 为什么要学 Hooks？

React Hooks 彻底改变了我们编写组件的方式。从 class 组件的 `this.setState` 到函数组件的 `useState`，代码变得更简洁、更可预测。

本文会结合实际场景，带你掌握最常用的 Hooks。

## useState：状态管理的基石

```tsx
import { useState } from "react";

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <button onClick={() => setCount((c) => c + 1)}>
      点击了 {count} 次
    </button>
  );
}
```

> **注意**：永远使用函数式更新 `setCount(c => c + 1)` 而不是 `setCount(count + 1)`，避免闭包陷阱。

### 复杂状态用 useReducer

当状态逻辑变复杂时，`useReducer` 是更好的选择：

```typescript
type Action =
  | { type: "increment" }
  | { type: "decrement" }
  | { type: "reset"; payload: number };

function reducer(state: number, action: Action): number {
  switch (action.type) {
    case "increment":
      return state + 1;
    case "decrement":
      return state - 1;
    case "reset":
      return action.payload;
  }
}

function Counter() {
  const [count, dispatch] = useReducer(reducer, 0);
  // dispatch({ type: "increment" })
}
```

## useEffect：副作用的正确姿势

`useEffect` 是最容易用错的 Hook。记住一个原则：**每个 effect 只做一件事**。

```tsx
function UserProfile({ userId }: { userId: string }) {
  const [user, setUser] = useState<User | null>(null);

  useEffect(() => {
    // ✅ 使用 AbortController 处理竞态
    const controller = new AbortController();

    fetch(`/api/users/${userId}`, { signal: controller.signal })
      .then((res) => res.json())
      .then(setUser)
      .catch((err) => {
        if (err.name !== "AbortError") console.error(err);
      });

    // 清理函数：组件卸载或 userId 变化时取消请求
    return () => controller.abort();
  }, [userId]);

  if (!user) return <p>加载中...</p>;
  return <h1>{user.name}</h1>;
}
```

### 常见错误

```tsx
// ❌ 对象依赖导致无限循环
useEffect(() => {
  fetchData(options);
}, [options]); // options 每次渲染都是新对象！

// ✅ 用 useMemo 稳定依赖
const stableOptions = useMemo(() => ({ page, limit }), [page, limit]);
useEffect(() => {
  fetchData(stableOptions);
}, [stableOptions]);
```

## useRef：不触发重渲染的"记忆"

`useRef` 有两个核心用途：

### 1. 访问 DOM 元素

```tsx
function AutoFocusInput() {
  const inputRef = useRef<HTMLInputElement>(null);

  useEffect(() => {
    inputRef.current?.focus();
  }, []);

  return <input ref={inputRef} placeholder="自动聚焦" />;
}
```

### 2. 存储可变值

```tsx
function Timer() {
  const intervalRef = useRef<ReturnType<typeof setInterval>>();

  const start = () => {
    intervalRef.current = setInterval(() => {
      console.log("tick");
    }, 1000);
  };

  const stop = () => {
    clearInterval(intervalRef.current);
  };

  return (
    <>
      <button onClick={start}>开始</button>
      <button onClick={stop}>停止</button>
    </>
  );
}
```

## 自定义 Hook：复用逻辑的利器

自定义 Hook 是 React 最强大的模式之一。它让你把组件逻辑抽象成可复用的函数。

### useLocalStorage

```typescript
function useLocalStorage<T>(key: string, initialValue: T) {
  const [stored, setStored] = useState<T>(() => {
    try {
      const item = localStorage.getItem(key);
      return item ? (JSON.parse(item) as T) : initialValue;
    } catch {
      return initialValue;
    }
  });

  const setValue = (value: T | ((val: T) => T)) => {
    const valueToStore = value instanceof Function ? value(stored) : value;
    setStored(valueToStore);
    localStorage.setItem(key, JSON.stringify(valueToStore));
  };

  return [stored, setValue] as const;
}

// 使用
const [theme, setTheme] = useLocalStorage("theme", "light");
```

### useDebounce

```typescript
function useDebounce<T>(value: T, delay: number): T {
  const [debounced, setDebounced] = useState(value);

  useEffect(() => {
    const timer = setTimeout(() => setDebounced(value), delay);
    return () => clearTimeout(timer);
  }, [value, delay]);

  return debounced;
}

// 在搜索框中使用
function SearchBar() {
  const [query, setQuery] = useState("");
  const debouncedQuery = useDebounce(query, 300);

  useEffect(() => {
    if (debouncedQuery) {
      searchAPI(debouncedQuery);
    }
  }, [debouncedQuery]);

  return <input value={query} onChange={(e) => setQuery(e.target.value)} />;
}
```

## React 19 的新 Hooks

React 19 带来了几个实用的新 Hook：

```tsx
// useActionState — 处理表单提交状态
import { useActionState } from "react";

function LoginForm() {
  const [state, formAction, isPending] = useActionState(
    async (prev: string, formData: FormData) => {
      const result = await login(formData);
      return result.error ?? "";
    },
    ""
  );

  return (
    <form action={formAction}>
      <input name="email" type="email" />
      <button disabled={isPending}>
        {isPending ? "登录中..." : "登录"}
      </button>
      {state && <p className="error">{state}</p>}
    </form>
  );
}
```

## 总结

| Hook | 用途 | 触发重渲染？ |
|------|------|:---:|
| `useState` | 管理状态 | ✅ |
| `useReducer` | 复杂状态逻辑 | ✅ |
| `useEffect` | 副作用（数据获取、订阅） | — |
| `useRef` | DOM 引用 / 可变值 | ❌ |
| `useMemo` | 缓存计算结果 | — |
| `useCallback` | 缓存函数引用 | — |

记住：Hooks 的核心理念是**组合**。把复杂逻辑拆分成小的自定义 Hook，然后像乐高积木一样组合起来。
